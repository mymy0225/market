# market
free market web application made with java (Spring Boot)h1

## 内容
フリーマーケットサイト<br>
Java、html、css、MySQLにて制作（Spring Bootを使用）<br>
機能は以下の通り
- ユーザー登録
- ログイン
- ログアウト
- ユーザー情報の編集
- 商品の出品
- 商品の編集
- お気に入り登録　他


## 注意
MySQLの実装されたローカル環境で動作確認しているため、実行にはMySQLのインストールおよび market\src\main\resources\application.properties 内の「 *** 」部分を環境に合わせて変更する必要があります。
